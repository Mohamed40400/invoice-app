# Invoice App

Gestion de Factures – Application Angulars
Description :

Cette application web sert à gérer les clients, les produits et les factures. Elle est créée avec Angular et utilise IndexedDB pour enregistrer les données localement sur le navigateur.

L’application permet de faire les actions suivantes :

- Ajouter, modifier et supprimer des clients.
- Gérer les produits avec leur nom, leur prix et la quantité disponible en stock.
- Créer des factures avec un calcul automatique du HT, de la TVA (10%) et du TTC.
- Utiliser une interface simple, responsive et bien présentée grâce à Bootstrap .
